Clearance — Human Approval API.

The approval layer for agent commerce.

When an agent needs to buy something, send something, or trigger a sensitive action, it asks for human sign-off first. Approved actions return a signed clearance token any downstream service can verify. Clean boundary between AI intent and real-world consequences.

  • One-tap human approval — works on any device
  • Signed JWT clearance tokens, downstream-verifiable
  • USDC on Base payment rail, on-chain verified
  • Free tier live · Pro tier $19/mo · Scale tier $49/mo
  • FastAPI backend, SQLite state, deployed on Railway

Orchestrate your agent fleet. Approve every move.

Felipe — Domain Acquisition Lab.

An approval-gated research tool for sourcing and valuing domain acquisitions.

Felipe is an internal acquisition research platform built on Python + FastAPI. It watches auctions, scores candidates against signals, proposes acquisitions, and routes higher-stakes buys through Clearance for explicit human approval.

  • Auction monitoring + value models for GoDaddy, Sedo, and DropCatch
  • Signal stack: SEO metrics, keyword demand, comparable sales
  • Clearance-integrated approval flow for sensitive-threshold purchases
  • Python · FastAPI · SQLite + optional Postgres · internal lab use
